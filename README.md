# Alum Tracker

## Description

This app tracks which Prime Digital Academy alum were present at different alumni events. New cohorts can be added with their graduation dates. Alum can be added to each event they attended.

## Technologies Used

- React
- React-Redux
- Firebase
- CSS
- HTML
